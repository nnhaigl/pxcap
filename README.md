# PIXCAP
## Dependencies
 * NodeJS
 * Express
 * Typescript
 * Mongo

## How to build
### Manual
- Install packages: `npm install`
- Setup env variable from `.env.example`
- Start by dev : `npm run dev`

### Docker Compose
Run script to start app by docker: `docker-compose up`
start app at `http://localhost:5555`

### Swagger:
`http://localhost:5555/swagger`
